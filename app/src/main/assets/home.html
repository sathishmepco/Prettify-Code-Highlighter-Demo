Sample Java Program


public class JavaMain {

    public static void main(String[] args) throws Exception{

        int i = 5;
        int j = 10;

        i += 5; //same as i = i + 5
        j -= 2; //same as j = j - 2

        System.out.println("i = " + i);
        System.out.println("j = " + j);

    }
}

Output:
i = 10
j = 8
